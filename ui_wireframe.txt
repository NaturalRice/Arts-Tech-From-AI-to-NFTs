UI Wireframe for NFT-Integrated Game:

1. [HUD]
   - Top Bar: Health Bar | Coin Counter | NFT Inventory Icon
   - Bottom Left: Quest Log Button
   - Bottom Right: Settings Button

2. [Main Menu]
   - Start Game
   - Load Game
   - Settings
   - Exit

3. [Quest Log]
   - List of Active Quests (Title, Description, Progress)
   - Reward Preview

4. [NFT Marketplace]
   - Display Owned NFTs
   - "Sell" or "Trade" Options

5. [Dialogue Box]
   - NPC Image
   - Text Area
   - Response Options
